## 1. 用户故事

### Epic 1: 项目管理

#### US-1.1 创建项目

```
作为 超级管理员
我想要 创建新项目并设置项目前缀
以便于 隔离不同产品的测试资产

- 输入项目名称（必填，全局唯一）

```
作为 管理员
我想要 在项目下创建功能模块并设置模块缩写

验收标准：
- 模块名称在项目内唯一
- 模块缩写在项目内唯一（2-4位大写字母）
- 删除模块时提示将删除关联用例
- 返回错误码 20004/20009 表示重复
```

```

### 2.2 知识库模块

#### 2.2.1 文档管理

| 功能 | API | 说明 |
|------|-----|------|
| 上传文档 | `POST /projects/{id}/documents` | multipart/form-data，支持 file 或 url |
| 获取文档详情 | `GET /documents/{id}` | 包含状态、分块信息 |
| 删除文档 | `DELETE /documents/{id}` | 软删除，级联删除分块 |
| 获取文档分块 | `GET /documents/{id}/chunks` | 分块列表 |

#### 2.2.2 文档状态流转

```
pending → processing → completed
                    → failed
```

| 状态 | 说明 |
|------|------|
| pending | 等待处理 |
| processing | 正在解析 |
| completed | 处理完成 |
| failed | 处理失败（记录 error_msg） |

#### 2.2.3 文档处理流程

```
上传文件 → 存储到 OSS → 创建 Document 记录（status: pending）
         ↓
    推送到 RabbitMQ
         ↓
    消费者解析文档 → 生成分块（DocumentChunk）
         ↓
    调用 Embedding API → 存储向量到 Milvus
         ↓
    更新 Document 状态为 completed
```

### 2.3 AI 用例生成模块

#### 2.3.1 生成任务

| 功能 | API | 说明 |
|------|-----|------|
| 创建生成任务 | `POST /projects/{id}/generation/tasks` | 异步处理 |
| 获取任务状态 | `GET /generation/tasks/{id}` | 包含草稿列表 |
| 获取草稿列表 | `GET /generation/drafts` | 支持按项目、状态、置信度筛选 |
| 保存草稿修改 | `PUT /generation/drafts/{id}` | 编辑草稿内容 |
| 确认草稿 | `PUT /generation/drafts/{id}/confirm` | 转为正式用例 |
| 拒绝草稿 | `PUT /generation/drafts/{id}/reject` | 记录反馈 |

#### 2.3.2 生成任务参数

```json
{
  "moduleId": "uuid",
  "prompt": "需求描述，最少20字",
  "documentScope": "all | prd_only | figma_only",
  "caseCount": 5,
  "sceneTypes": ["positive", "negative", "boundary"],
  "priorityPreference": "P2",
  "generationMode": "normal | deep",
  "documentIds": ["uuid"]
}
```

#### 2.3.3 RAG 检索流程

```
用户需求描述
     ↓
调用 Embedding API 转为向量
     ↓
在 Milvus 中检索 Top-K 相似文档块
     ↓
组装 Prompt（需求 + 检索上下文）
     ↓
调用 LLM 生成用例 JSON
     ↓
解析并存入 generated_case_draft 表
```

#### 2.3.4 置信度计算规则

| 级别 | 判定条件 | UI 展示 |
|------|----------|---------|
| high | 检索到 ≥2 个文档块，语义相似度 > 0.8 | 🟢 绿色 |
| medium | 检索到 1 个文档块，语义相似度 0.5-0.8 | 🟡 黄色 |
| low | 无匹配文档块，纯 LLM 生成 | 🔴 红色 |

#### 2.3.5 降级策略

| 场景 | 处理方式 |
|------|----------|
| 检索结果 < 1 个文档块 | 弹窗提示，用户确认后继续，置信度标记为 low |
| 完全无知识库 | 禁止发起，返回错误码 30007 |

### 2.4 测试用例管理模块

#### 2.4.1 用例属性

| 属性 | 类型 | 必填 | 说明 |
|------|------|:----:|------|
| number | String | 自动 | 业务编号 |
| title | String | ✅ | 用例标题 |
| preconditions | JSON Array | | 前置条件 |
| steps | JSON Array | ✅ | 测试步骤 |
| expected | JSON Object | ✅ | 预期结果 |
| caseType | Enum | | functionality/performance/api/ui/security |
| priority | Enum | | P0/P1/P2/P3 |
| status | Enum | 自动 | unexecuted/pass/block/fail |
| aiMetadata | JSON Object | | AI 生成信息 |

#### 2.4.2 用例编号规则

**格式**：`{项目前缀}-{模块缩写}-{日期}-{序号}`

**示例**：`ECO-USR-20260402-001`

**组成**：
- 项目前缀：创建项目时输入，2-4 位大写字母，全局唯一
- 模块缩写：创建模块时输入，2-4 位大写字母，项目内唯一
- 日期：YYYYMMDD 格式
- 序号：当日该模块下 001-999 递增

#### 2.4.3 AI 元数据结构

```json
{
  "generation_task_id": "uuid",
  "confidence": "high | medium | low",
  "referenced_chunks": [
    {
      "chunk_id": "uuid",
      "document_id": "uuid",
      "document_title": "登录模块PRD",
      "similarity_score": 0.85
    }
  ],
  "model_version": "deepseek-chat-v3",
  "generated_at": "2026-04-02T10:00:00Z"
}
```

## 3. 非功能性需求

### 3.1 性能要求

| 指标 | 目标值 | 验证方式 |
|------|--------|----------|
| API 响应时间 | P99 < 500ms | k6 负载测试 |
| 用例列表加载 | < 2s (1000条) | 前端性能监控 |
| 文档解析 | < 30s (50页PDF) | 端到端测试 |
| AI 生成任务 | < 60s (5个用例) | 端到端测试 |
| 向量检索 | P99 < 500ms | Milvus 基准测试 |

### 3.2 LLM 调用超时配置

| 配置项 | 超时时间 | 说明 |
|--------|----------|------|
| LLM 单次调用 | 30s | 大模型推理超时 |
| 总任务超时 | 120s | 含检索 + 生成全流程 |
| Embedding 调用 | 10s | 向量化超时 |

### 3.3 失败处理策略

| 失败类型 | 处理方式 | 错误码 |
|----------|----------|--------|
| LLM 超时 | 任务状态 failed，提示重试 | 60010 |
| 模型服务异常 | 记录日志，提示稍后重试 | 60007 |
| Embedding 失败 | 降级为无 RAG，置信度 low | 30005 |
| 队列积压 | 显示预计等待时间 | 60011 |

### 3.4 安全要求

| 要求 | 实现方式 |
|------|----------|
| 认证 | JWT Token，有效期 2 小时 |
| 密码存储 | bcrypt 加密 |
| 敏感配置 | AES-256 加密存储 |
| 访问控制 | RBAC（3 角色） |
| Figma Token | 必须加密存储 |
| 审计日志 | 敏感操作记录，保留 ≥180 天 |

### 3.5 审计日志记录的操作

| 操作类别 | 具体操作 |
|----------|----------|
| 认证 | 登录、登出、密码修改 |
| 项目管理 | 创建、删除项目 |
| 配置管理 | 新增、修改、删除配置 |
| 文档管理 | 上传、删除文档 |
| AI 生成 | 发起生成任务 |
| 用例管理 | 创建、删除用例（批量） |
| 执行管理 | 录入执行结果 |

### 3.6 监控告警规则

| 告警名称 | 触发条件 | 级别 |
|----------|----------|------|
| API 错误率过高 | 5xx 错误率 > 1% 持续 5 分钟 | 🔴 严重 |
| API 响应过慢 | P99 延迟 > 1s 持续 5 分钟 | 🟡 警告 |
| LLM 调用失败率过高 | 失败率 > 5% 持续 10 分钟 | 🔴 严重 |
| 文档解析队列积压 | 待处理任务 > 100 | 🟡 警告 |
| 生成任务超时率过高 | 超时率 > 10% 持续 10 分钟 | 🟡 警告 |
| 数据库连接池耗尽 | 空闲连接 < 2 | 🔴 严重 |
